Sistema Decimal

O sistema decimal é um sistema de numeração de base 10, formado pelos dígitos de 0 a 9. Ele é considerado um sistema posicional, ou seja, o valor de cada dígito depende da sua posição no número. Cada casa representa uma potência de 10, da direita para a esquerda (unidades, dezenas, centenas, etc.). Por exemplo, o número 3742 representa 3×10³ + 7×10² + 4×10¹ + 2×10⁰. O sistema decimal é amplamente utilizado no cotidiano e em diversas áreas do conhecimento, como matemática, finanças e ciências, por ser intuitivo e compatível com o modo natural de contagem humana, baseado em dez dedos. Sua estrutura facilita cálculos e a representação de quantidades, tornando-o um padrão universal na comunicação numérica.

Sistema Binário

O sistema binário é um sistema de numeração de base 2, que utiliza apenas dois dígitos: 0 e 1. Assim como o sistema decimal, ele também é posicional, ou seja, o valor de cada dígito depende de sua posição e é multiplicado por potências de 2. Por exemplo, o número binário 1011 representa 1×2³ + 0×2² + 1×2¹ + 1×2⁰, resultando no número decimal 11. O sistema binário é fundamental para a computação e a eletrônica digital, pois os computadores funcionam com dois estados básicos (ligado e desligado), que são representados por 1 e 0. Sua simplicidade permite o processamento rápido e eficiente de informações, sendo a base de toda a lógica dos sistemas digitais modernos.

Sistema Hexadecimal

O sistema hexadecimal é um sistema de numeração de base 16, que utiliza dezesseis símbolos para representar os valores: os dígitos de 0 a 9 e as letras de A a F, onde A representa 10, B representa 11, e assim por diante até F, que representa 15. Assim como os sistemas decimal e binário, o hexadecimal também é posicional, e cada posição representa uma potência de 16. Por exemplo, o número hexadecimal 2F representa 2×16¹ + 15×16⁰, resultando em 47 no sistema decimal. O sistema hexadecimal é muito utilizado em computação e programação, pois permite representar grandes valores binários de forma mais compacta e legível, já que um único dígito hexadecimal equivale a quatro dígitos binários. Por isso, ele é frequentemente usado para endereços de memória, cores em HTML/CSS e instruções em baixo nível.

Sistema Octal

O sistema octal é um sistema de numeração de base 8, que utiliza apenas os dígitos de 0 a 7 para representar os valores. Assim como os sistemas decimal, binário e hexadecimal, octal é um sistema posicional, onde cada posição representa uma potência de 8. Por exemplo, o número octal 157 equivale a 1×8² + 5×8¹ + 7×8⁰, o que resulta em 111 no sistema decimal. O sistema octal é utilizado principalmente na área da computação, especialmente em sistemas embarcados e antigos sistemas operacionais, por ser mais compacto que o binário. Como cada dígito octal equivale a três dígitos binários, ele serve como uma forma simplificada de representar códigos binários longos. Embora atualmente o hexadecimal seja mais comum, o octal ainda é útil em certos contextos técnicos e educacionais.
